# Linux Configuration

## Recommended packages

```
# Sound server
sudo pacman -S pulseaudio pulseaudio-alsa pulseaudio-bluetooth pulseaudio-equalizer

# Clipboard manager
sudo pacman -S cliphist

# Backlight control
sudo pacman -S brightnessctl

# Trash management
sudo pacman -S trash-cli

# Auto mount tool
sudo pacman -S udisks2 udiskie

# Media player control
sudo pacman -S playerctl

# Screenshot tool
sudo pacman -S grim slurp swappy

# Image viewer
sudo pacman -S imv
```
